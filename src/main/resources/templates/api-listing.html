Welcome to fastub (v1.0.0)
Get ready API'S in seconds

Fastub is a easiest way to develop API with mock data.

Below are some cool features of fastub.

  • Fastub support all http response codes (200(OK),400(Bad request) etc.)
  • You can mock request using different http methods. Currently it supports five http methods (GET,POST,PUT,PATCH,DELETE);
  • You can configure multiple response body for one API request.
  • You can mock response headers using fastub.
